Voice Reco

Voice Reco is a Speech Recognition web app. It respond to the sounds produced by human speech and transcripts into text.
This app supports multiple languages: English, French, Hindi, Urdu, etc.

Voice Commands

  • CUT CUT Delete recently entered text.
  • NEXT LINE Add full stop and move to new line.
  • FULL STOP Add full stop.
  • COPY TEXT Copy to clipborad as Plain Text.
  • COPY HTML Copy to clipborad as HTML.

*Note: Voice commands work with English language only.
